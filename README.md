# My About Me App

A personal website that serves as an online resume and link tracker, with an integrated Jellyfin media server page. Built with plain HTML, CSS, and JavaScript — no frameworks, no build step — and deployed via Netlify.

---

## Live Site

[your-site-name.netlify.app](https://your-site-name.netlify.app)

---

## Features

- **About Me** — Bio, skills, experience, and contact section
- **Links** — Quick access to GitHub, LinkedIn, email, and more
- **Media Server** — Jellyfin library stats and recently added media, powered by a Python data fetch script

---

## Built With

- HTML, CSS, JavaScript
- Python 3 (local data fetch script)
- [Jellyfin](https://jellyfin.org) — open source media server
- [Tailscale](https://tailscale.com) — private network access to home server
- [Netlify](https://netlify.com) — static site hosting with automatic Git deploys

## Project Structure

```
my-about-me-app/
├── index.html            # Main About Me page
├── jellyfin.html         # Media server page
├── netlify.toml          # Netlify deploy config
├── data/
│   └── jellyfin.json     # Static media data (generated by Python script)
├── css/
│   ├── style.css         # Shared styles
│   └── jellyfin.css      # Jellyfin page styles
└── js/
    ├── main.js           # Scroll animations, shared JS
    └── jellyfin.js       # Reads jellyfin.json and renders the media page
```

> **Note:** `fetch_jellyfin.py` is intentionally excluded from this repository
> as it contains private API credentials. It runs locally to generate `data/jellyfin.json`.

## Running Locally

1. Clone the repo
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
   cd my-about-me-app
   ```

2. Start a local web server
   ```bash
   python -m http.server 8080
   ```

3. Open your browser to `http://localhost:8080`

## Updating Jellyfin Data

The media page is powered by a static JSON file generated by a local Python script.
To refresh your media data:

1. Connect to Tailscale (if away from home)
2. Activate your virtual environment
   ```bash
   .venv\Scripts\activate        # Windows
   source .venv/bin/activate     # Mac/Linux
   ```
3. Run the fetch script
   ```bash
   python fetch_jellyfin.py
   ```
4. Commit and push the updated JSON
   ```bash
   git add data/jellyfin.json
   git commit -m "Update Jellyfin data"
   git push
   ```

Netlify will auto-deploy within ~30 seconds.
